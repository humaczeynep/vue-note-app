# 📝 Vue Note App

Bu proje, Vue 3 ve Pinia kullanılarak geliştirilmiş basit bir not uygulamasıdır.

## 🚀 Özellikler

- Not ekleme
- Not silme
- Not düzenleme
- Kategoriye göre filtreleme
- Önceliğe göre sıralama
- Başlık ve içerik bazlı arama
- Dashboard (istatistikler)

## ⚙️ Kurulum

Projeyi çalıştırmak için:

```bash
git clone https://github.com/humaczeynep/vue-note-app.git
cd vue-note-app
npm install
npm run dev
=======
